Triangulator

A visual triangulation tool

Instructions

To use this tool, you will need the Cell Tower ID (CID) and Location Area Code (LAC) from three cell towers, obtainable through any iOS or Android cellular device with the OpenSignal app installed.

Using the CID and LAC of these three cell towers, this tool will calculate the estimated location of the cellular device and display the results on an interactive map.

As there is no publicly-available API to extract an accurate signal strength from the cellular device, this tool assumes that the strength to all three cell towers is equal.

Contribute

This tool is open-sourced on GitHub.

If you find any bugs or would like to suggest improvements, do feel free to open an issue or send us a pull request there.

Authors
